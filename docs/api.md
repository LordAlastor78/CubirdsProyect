# API y clases principales

Esta página lista las clases principales y una breve descripción. Para ver el código fuente, consultar los ficheros en `gal.uvigo.esei.aed1_cubirds/src/main/java`.

- `Card` — representa una carta del juego: especie, contadores y umbrales.
- `DeckOfCards` — implementa la baraja, mezclar y robar cartas.
- `Game` — lógica del flujo de la partida, turnos y condiciones de fin.
- `Player` — estado del jugador (mano, zona de juego, contadores).
- `Table` — filas en mesa y operaciones de inserción/robo.
- `TypeBird` — enumeración de especies disponibles.
- `IU` — utilidades de interfaz/entrada salida usadas por `Main`.
- `Main` — punto de entrada para ejecutar una partida de prueba.
