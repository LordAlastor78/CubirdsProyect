# Descripción general

Proyecto: CuBirds — implementación en Java del juego de cartas para la asignatura AED I.

Objetivo del repositorio:

- Implementar la lógica del juego (modelo) en el paquete `gal.uvigo.esei.aed1.cubirds.core`.
- Proveer una interfaz mínima en `gal.uvigo.esei.aed1.cubirds.iu` para ejecutar partidas.

Alcance de esta documentación:

- Explicar cómo compilar y ejecutar la aplicación.
- Describir la estructura del proyecto y las clases principales.
- Ofrecer referencias rápidas a los ficheros fuente.
